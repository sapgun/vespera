# VESPERA

**Vision Enhanced Sovereign Personal Execution & Resource Architecture**

> The evening star of your AI sovereignty.

VESPERA is a local-first AI operations kit that lets you own your knowledge, creative assets, approvals, and multi-agent project workflows — while treating AI tools as workers, not owners.

## Why VESPERA?

Modern AI workflows are powerful but fragmented.  
Conversations, decisions, generated assets, and approvals scatter across ChatGPT, Claude, Grok, Runway, Kling, n8n, and more.

VESPERA brings everything back under **your sovereignty** — in your own storage, with your own rules.

## Core Philosophy

1. AI tools are workers, not owners.
2. Markdown is the source of truth (with flexible HTML output support).
3. Creative assets must be recovered from AI platforms.
4. Automation starts with human approval.
5. Local-first is the default.
6. Final authority belongs to the user.

## What VESPERA Includes (v0.1)

- Obsidian Vault structure + core templates
- Asset Library (local / Google Drive / NAS / S3 ready)
- AI Team Roles (`config/agents.yaml`)
- Permission Matrix + Approval Queue
- Aether Crew Lite project template
- install.ps1, healthcheck.ps1, backup.ps1
- Full documentation (`docs/`)

## Quick Start (5 minutes)

1. Run `.\scripts\install.ps1`
2. Open the VESPERA Vault in Obsidian
3. Start your first project in `01_Projects\Aether_Crew_Lite`
