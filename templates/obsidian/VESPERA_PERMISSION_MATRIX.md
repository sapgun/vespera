# VESPERA Permission Matrix

## Level 0 — Free
- 노트 요약, 아이디어 생성, 태그 추천

## Level 1 — Log Only
- draft note 생성, daily brief 저장

## Level 2 — Human Approval Required
- 파일 이동/이름 변경, GitHub Issue 생성, Asset Final 표시

## Level 3 — Double Confirmation
- 콘텐츠 발행, PR 생성, 파일 삭제

## Level 4 — Manual Only
- seed phrase, private key, wallet signing, payment
