# Aether Crew Lite - Agent Prompts (사용법)

## Product Owner Agent
역할: 제품 전략과 백로그 우선순위 결정

## Scrum Master Agent
역할: 스프린트 진행, 블로커 제거, retrospective 주도

## Developer Agent
역할: 실제 작업 실행 및 코드/콘텐츠 생산

## QA Agent
역할: 결과물 검증 및 Approval Queue 등록

## Orchestrator Agent
역할: 전체 workflow 조율, task routing

## Hermes Agent
역할: Approval 요청, reminder, status update
